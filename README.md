# 🚀 Frontend Mini Projects Collection

This repository contains **two modern web projects** built using **HTML, CSS, and JavaScript**.

Both projects are designed with a clean UI, responsive layout, and interactive features.

---

## 📁 Projects Included

### 1️⃣ Modern Login and Registration System
A stylish authentication system with login, registration, and dashboard pages.

**Features**
- User Login
- New User Registration
- Show / Hide Password
- Remember Me Option
- localStorage Data Storage
- Welcome Dashboard
- Logout Functionality
- Modern Glassmorphism UI

**Files**
```text
login-system/
│
├── index.html
├── register.html
├── dashboard.html
├── style.css
└── script.js
```

---

### 2️⃣ Nexus OS | Task Architecture
A futuristic task manager and reminder application.

**Features**
- Live Clock
- Live Date
- Add New Tasks
- Emoji Categories
- Set Date and Time
- Reminder Toggle
- Delete Task
- Task Counter
- localStorage Support
- Modern Dark UI

**Files**
```text
nexus-task-manager/
│
├── index.html
├── style.css
├── script.js
└── README.md
```
